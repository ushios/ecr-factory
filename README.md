# ecr-factory

AWS ECR factory

# Installation

```console
$ go get github.com/ushios/ecr-factory
```

# Usage

Create repository and policy that are pull and push.
```console
$ ecrf -id xxxxx -secret xxxx -name your_docker_repository_name
```
